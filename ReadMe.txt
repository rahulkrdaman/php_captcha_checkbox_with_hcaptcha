

============ Instructions ============
1. Register your site at hCAPTCHA site - https://hCaptcha.com

2. Get your Site Key and Secret Key.

3. Open the submit.php file on editor ===> Specify the Site Key ($siteKey) and Secret Key ($secretKey). Also, specify the email address of the site admin in the $to variable.

4. Open the index.php file on the browser and test the CAPTCHA functionality.
